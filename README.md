# 사다리 게임
 ## 사용자 입력 처리
  * 인원수를 입력받는다. 
  * 인원수 만큼 각 사용자의 이름을 입력받는다. 
  * _예외처리_:
       1. 입력한 인원수가 숫자가 아닌 경우
       2. 입력한 사용자의 이름 개수가 인원수 보다 작거나 많을 경우
       3. 입력한 사용자의 이름이 5글자 이하가 아닌 경우
       4. 입력한 사용자의 이름이 any, all, every 와 같은 불특정 다수를 의미하는 단어일 경우
       *  1 -> 인원수 다시 입력
       * 2, 3, 4 -> 이름 다시 입력받기
       
 ## 사다리 만들기 
   1. 사용자 수 만큼의 기둥들을 만든다.
   2. 인접한 두 기둥을 연결짓는 다리 갯수를 랜덤으로 입력받는다.  
   3. 2에서 입력받은 다리 개수 만큼 인접한 두 기둥들을 연결한다. 
   4. 3에서 두 다리가 인접하지 않도록 한다. 
 ## 만든 사다리로 게임 진행해서 순위 매기기
    1. 만든 사다리로 순위를 매긴다. 
 ## 순위 출력
  * 매겨진 순위를 출력한다.
